# Attention-Enhanced SqueezeNet for Maize Disease Detection

This repository contains the implementation of an attention-enhanced SqueezeNet model for robust maize leaf disease detection under real-world agricultural conditions.

## Objectives

* Implement baseline SqueezeNet using PyTorch
* Integrate Squeeze-and-Excitation (SE) attention blocks
* Evaluate robustness on real-world maize leaf datasets
* Optimize the model for mobile deployment

## Tools

* PyTorch
* GitHub
* Python
* Android Deployment

## Research Focus

Improving lightweight convolutional neural networks for precision agriculture applications while maintaining mobile efficiency.
